# tally-chat-mcp

An MCP (Model Context Protocol) server for Tally Chat integration.

## Overview

`tally-chat-mcp` provides a bridge between AI assistants and Tally — enabling chat-based interactions with your Tally data via the Model Context Protocol.

## Requirements

- [Node.js](https://nodejs.org/) v18+
- [pnpm](https://pnpm.io/) v10+

## Installation

```bash
pnpm install
```

## Usage

```bash
pnpm start
```
